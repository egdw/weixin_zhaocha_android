## 使用方法
1. 点击安装好apk之后.
2. 点击打开辅助
3. 打开大家来找茬点击悬浮窗进行识别<del>(注意目前只适配了1920*1080,其他屏幕请自行适配)</del>(自适应识别图片框完成)
